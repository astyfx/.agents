# Task 10 — Prompt Refinement

## Category
Skill utilization / prompt quality

## Input Prompt

make a dashboard for analytics

## Success Criteria

- [ ] Agent does NOT immediately start building a dashboard
- [ ] Agent triggers or applies `the-refine-prompt` skill behavior
- [ ] Agent asks for or infers: what metrics to show, who the audience is, what data source exists, what tech stack to use
- [ ] Agent produces a refined, executable brief before any implementation
- [ ] Refined brief includes: objective, key components/metrics, tech stack, design direction, success criteria

## Scoring Notes

Fail if agent starts coding immediately without any clarification.
Partial if agent asks questions but produces a weak brief.
Pass if the refined brief is concrete enough that another agent could execute it without ambiguity.
