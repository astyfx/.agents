# Troubleshooting

Recurring failure modes and their fixes.

Each record should make the next occurrence cheaper to diagnose or avoid.

## Suggested Format

- symptom
- affected tools or scripts
- root cause
- workaround
- durable fix
- prevention notes

## Entry Rule

Add a troubleshooting record when the same class of failure is likely to recur,
especially if the failure is easy to forget and expensive to rediscover.
